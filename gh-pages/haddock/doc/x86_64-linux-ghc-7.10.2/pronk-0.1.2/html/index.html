pronk-0.1.2: A library and command line app for HTTP load testing

A library and command line tool for HTTP load testing. You can think of this package as similar to httperf or ab, only more modern, faster, more scalable, easier to deal with, programmable, and with customisable output.

The name "pronk" comes from the Afrikaans verb pronk-, meaning "to show off" or "to strut".
