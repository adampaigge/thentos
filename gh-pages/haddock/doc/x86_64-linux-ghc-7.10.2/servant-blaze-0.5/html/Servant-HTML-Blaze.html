Servant.HTML.Blaze

Description

An HTML empty data type with MimeRender instances for blaze-html's ToMarkup class and Html datatype. You should only need to import this module for it's instances and the HTML datatype.:

>>> type Eg = Get '[HTML] a

Will then check that a has a ToMarkup instance, or is Html.
